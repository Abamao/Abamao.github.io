## 🛠️ 技术栈

- **前端**: 纯 HTML5 + CSS3 + JavaScript
- **样式**: CSS 变量主题系统、Flexbox/Grid 布局
- **动画**: CSS Keyframes、JavaScript 动画
- **存储**: LocalStorage 本地存储
- **部署**: GitHub Pages
